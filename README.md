# generator-grunt-awesome-plugin
A Yeoman generator for Grunt plugin, the awesome way

## Install

### Yeoman

```sh
npm install -g yo
```

### This generator

```sh
npm install -g generator-grunt-awesome-plugin
```

## Create a brand new Grunt plugin

```sh
mkdir grunt-moo
cd grunt-moo
yo grunt-awesome-plugin
```
